Comandos generales para el servidor de Django:
______________________________________________



	_python manage.py help

		_éste nos brinda una vista de ayuda donde muestra los comandos disponibles a utilizar dentro de la misma consola de comandos de window, 

		en la ruta de un proyecto ó app, 


		_python manage.py sql --help


			_más si requerimos información específica de uno de éstos comandos mostrados podemos tipear el siquiente código, ej 

			"python manage.py sql --help", así, éste nos arrojará información del comando "sql"
